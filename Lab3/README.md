# Лабораторная работа №3: Преобразование арифметических выражений с помощью деревьев

## Описание
Вариант 22
Программа выполняет преобразование арифметических выражений, представленных в виде бинарных деревьев. 
Реализация сложения и вычитания дробей по формуле:
$$a/b + c/d \to (a * d + b * c) / (b * d)$$

## Структура проекта
- `func.h` — Определение структур и прототипы функций
- `func.c` — Логика построения дерева, трансформации и вычислений
- `main.c` — Точка входа
- `test.c` — Набор тестов
- `Makefile` — Сценарий сборки проекта

## Сборка и запуск

### Основная программа
Для сборки и запуска введите в терминале:
```bash
make
make run
```

### Тестирование
Для запуска тестов:
```bash
make run_test
```

## Пример работы
**Ввод:** `1/2 + 1/3`
**Результат:** `(((1*3)+(2*1))/(2*3))`
