# Лабораторная работа №4: AVL-дерево

Реализация сбалансированного AVL-дерева на языке C. В качестве ключей используются строки латинских букв

## Структура проекта
- `func.h` — заголовочный файл с определениями структур и прототипов.
- `func.c` — реализация логики AVL-дерева (балансировка, повороты, вставка, удаление).
- `main.c` — интерактивный CLI для работы с деревом.
- `test.c` — модульные тесты (проверка балансировки и корректности операций).
- `Makefile` — скрипт для сборки проекта.

## Сборка и запуск

### Основная программа
Сборка и запуск интерактивного интерфейса:
```bash
make all
./main
```

### Команды CLI
В режиме `./main` доступны следующие команды:
- `i <key>` — вставить ключ (max 6 символов).
- `d <key>` — удалить ключ.
- `s <key>` — поиск ключа.
- `p` — вывод дерева (in-order) и его высоты.
- `q` — выход.

### Тестирование
Сборка и запуск автоматических тестов:
```bash
make run_test
```

## Особенности реализации
- Автоматическая балансировка при вставке и удалении.
- Лексикографическое (алфавитное) сравнение строковых ключей.
