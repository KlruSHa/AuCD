# Лабораторная работа: Деревья на языке Си

Программа для работы с двоичными деревьями поиска. Реализована на языке Си с разделением на заголовочные файлы и модули для удобства тестирования и сборки.

## Основные возможности
* **Двоичное дерево поиска (b_node):**
    * Добавление узлов с автоматическим сохранением порядка.
    * Удаление узлов с корректной обработкой сохранения порядка.
    * Функция find_min для поиска преемника при удалении.
    * Визуализация дерева в консоли с отступами, отражающими глубину узлов.
* **22 Вариант задания:**
    * Реализована функция get_deg, которая вычисляет степень дерева.

## Структура проекта
* tree.h — описания структур b_node, c_node и прототипы всех функций.
* func.c — логика работы с памятью, алгоритмы добавления, удаления и обхода.
* main.c — пользовательский интерфейс с текстовым меню.
* test.c — набор из 5 тестов, которые проверяют корректность функций из func.c
* Makefile — скрипт для автоматизации сборки проекта.

## Сборка и запуск

Проект собирается с помощью утилиты make.

1. Компиляция основной программы:
   ```
   make all
   ```

2. Запуск программы:
   ```
   make run
   ```

3. Запуск тестов:
   ```
   make run_test
   ```

4. Очистка проекта от объектных файлов:
   ```
   make clean
   ```

##  Технические особенности
* Управление памятью: Используются malloc и free. Реализована функция clear_b для полной очистки дерева перед завершением программы.
* Рекурсия: Все основные операции (печать, удаление, подсчет степени) выполнены с применением рекурсивных алгоритмов.
