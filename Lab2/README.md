# Лабораторная работа №2: Таблицы, Сортировка и Поиск

Данная лабораторная работа реализует простую систему управления таблицей пар "ключ-значение" на языке C. Основное внимание уделено алгоритмам сортировки методом двоичной вставки и бинарному поиску.

## Основные возможности

- Программа поддерживает работу со строками произвольной длины
- Сортировка двоичными вставками для ключей
- Бинарный поиск для поиска значений по ключу

## Структура проекта

- `table.h` — Определение структур `Item` (элемент таблицы) и `Table`, а также прототипы функци
- `table.c` — Реализация функций работы с таблицей 
- `main.c` — Точка входа в программу
- `tests.c` — Тесты для проверки корректности
- `data.txt` — Пример входных данных

## Сборка и запуск

### Сборка
Для компиляции проекта выполните команду:
```bash
make all
```

### Запуск основной программы
Вы можете запустить программу и вводить данные вручную или использовать перенаправление ввода из файла:
```bash
# Ручной ввод
./main

# Ввод из файла
./main < data.txt
```

### Запуск тестов
Для проверки работоспособности выполните:
```bash
make run_tests
```

## Пример работы

### Входные данные (`data.txt`)
Программа сначала ожидает количество элементов, затем пары ключ-значение, а в конце — ключи для поиска (до слова "exit").
```text
3
fruit
apple
color
red
animal
dog
animal
color
exit
```

### Вывод программы
```text
Table:
fruit
apple
color
red
animal
dog

Sorted table:
animal
dog
color
red
fruit
apple
Found: dog
Found: red
```
